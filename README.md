# Installing:
install [nodejs](https://nodejs.org/en/) if you didn't install it earlier.

`git clone https://github.com/Seothp/funbox-test-map.git`

`cd funbox-test-map`

`npm install`

## Start local server with project:
`npm start`

## Start cypress tests:
`npx cypress open`

## Start jest tests:
`npx run test`
